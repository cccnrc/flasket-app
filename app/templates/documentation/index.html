Flasket Installation


Flasket is a Flask-based app for ticketing system

Flasket can be deployed in two main ways:
  • integrate with your existing application: Flasket
  • standalone application: Flasket-App

    • 1.Flasket

    The advantage of Flasket is that you integrate this ticketing system with your existing application:
  • only registered users will be able to submit and interact with tickets, avoiding bots, spam, etc.
  • your website will serve the ticketing system as its own service, you are not redirected to a different website etc.
  • e-mail updates are sent to users for every ticket important activity, and these mails will come from your website, not external servers etc.
  • you can edit tickets as you wish to best reflect your application needs
  • extremely easy to integrate and update as it lives in a dedicated folder within your application

    • 2.Flasket-App

    The advantage of Flasket-App is that you deploy it as it comes and you are ready to go:
  • very little setup work
  • database, imports, etc. already configured
  • you can shape your application deployment around this "skeleton"
  • very detailed instructions for local and server setup
